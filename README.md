# Model-magazynu-wysokiego-sk-adowania-RFID
Prototypowy projekt automatycznej układnicy do magazynu wysokiego składowania. Do sterowania automatycznego wykorzystuje system RFID i mikrokontroler Arduino. Projekt został wykonany od podstaw i jest także moją pracą inżynierską.
